Instructions

  1. Use this page to generate official RINGO cards.
  2. Print the cards. They should be able to print two per page on a portrait-oriented letter paper. If they don't, please submit an issue below.
  3. Get in the car, start driving, and play RINGO as a fun family game, and use whichever playing style you want, such as:
    • Five in a row
    • Four corners
    • Blackout
    • Any other way you want...

Notes

  • At the center of every card, there is a free space marked as "Road".

Issues

If you find an issue with this program, please submit an issue on GitHub.
